# Mini Financial Dashboard & Personal Finance Simulator

An interactive web application built with **Python**, **Pandas**, and **Streamlit** designed to analyze stock market data and simulate personal financial decisions for young individuals. 

# Key Features
* **Historical Price Comparison:** Multi-stock selection to visualize asset performance.
* **Technical Indicators:** Implements Moving Averages (`MA7`/`MA30`).
* **Risk Assessment:** Calculates annualized volatility and 14-day RSI tracker.
* **Personal Finance Simulator:** Computes debt impact and interest weights relative to monthly income.

# Tech Stack
* Python, Pandas, Streamlit, and Plotly.
